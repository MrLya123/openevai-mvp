# OpenEVAI MVP 使用指南

## 📖 快速上手

### 首页功能

访问应用后，你会看到两个核心功能入口：

1. **📝 文生图** - 通过文字描述生成设计效果图
2. **🖼️ 图生图** - 上传草图/白模生成渲染图

---

## ✍️ 文生图使用教程

### 步骤 1: 选择类型

- **🏛️ 建筑设计**: 生成建筑外观、外立面、景观等
- **🏠 室内设计**: 生成室内空间、装修风格、软装搭配

### 步骤 2: 选择风格（可选）

#### 建筑风格预设
- **现代中式**: 简洁线条 + 传统元素现代化
- **极简主义**: 干净利落，几何形态
- **新古典**: 对称布局，精致装饰
- **当代风格**: 创新材料，玻璃幕墙
- **工业风**: 裸露结构，金属材质
- **地中海风格**: 白色墙面，蓝色点缀

#### 室内风格预设
- **现代轻奢**: 高级材质，低调奢华
- **北欧风**: 自然光线，木质元素
- **新中式**: 传统韵味，禅意空间
- **装饰艺术**: 几何图案，复古优雅
- **侘寂风**: 自然质朴，不完美之美
- **智能家居**: 科技感，自动化设备

### 步骤 3: 输入描述

**好的提示词 = 详细 + 具体 + 专业**

#### 优秀示例 ✅
```
高端现代中式风格的独栋别墅建筑，深灰色大理石外立面，
大面积落地窗，黄昏时分暖色灯光，8K 超高清，建筑摄影
```

```
北欧风格客厅，浅色橡木地板，米白色布艺沙发，
绿植点缀，自然光线从落地窗射入，温馨舒适
```

#### 避免的提示词 ❌
```
好看的房子
```
```
漂亮的客厅
```

### 步骤 4: 生成与下载

- 点击"✨ 开始生成"
- 等待 10-30 秒
- 下载图片或重新生成

## 🖼️ 图生图使用教程

### 适用场景

1. **手绘草图 → 效果图**: 快速可视化概念
2. **白模渲染 → 材质贴图**: 添加真实材质和光影
3. **线稿 → 彩色渲染**: 自动上色和细节丰富
4. **低质量图 → 高质量**: 提升图片质感

### 步骤 1: 上传图片

支持方式：
- 点击上传区域选择文件
- 直接拖拽图片到上传区域

**要求：**
- 格式：JPG, PNG
- 大小：< 10MB
- 建议分辨率：512x512 或更高

### 步骤 2: 选择渲染风格

- **写实渲染**: 照片级真实感
- **艺术表现**: 强调氛围和色彩
- **建筑表现**: 专业建筑摄影风格
- **概念设计**: 创意表达，独特视角

### 步骤 3: 补充描述（可选）

可以指定：
- 材质偏好（如"石材外立面"、"木质地板"）
- 颜色方案（如"暖色调"、"黑白灰"）
- 光线效果（如"自然光"、"黄昏"）
- 时间季节（如"春季"、"夜晚"）

### 步骤 4: 对比与下载

生成后会显示：
- **左侧**: 原始上传图
- **右侧**: AI 生成结果

可下载结果或重新渲染。

## 💡 提示词技巧

### 建筑结构描述
- 建筑类型：独栋别墅、高层公寓、商业综合体
- 层数：三层、十层、摩天大楼
- 形状：矩形、L 型、曲线形

### 材质描述
- 外立面：大理石、玻璃幕墙、清水混凝土、红砖
- 室内：实木地板、瓷砖、地毯、微水泥

### 光线描述
- 时间：清晨、正午、黄昏、夜晚
- 光源：自然光、射灯、灯带、吊灯
- 氛围：温暖、清冷、明亮、柔和

### 风格关键词
- 现代：简洁、流畅、几何
- 传统：对称、雕花、木质
- 自然：绿植、原木、石材
- 科技：金属、玻璃、LED

### 质量增强词
- 8K 超高清
- 照片级真实
- 专业渲染
- 建筑摄影
- 电影级灯光

## 📚 历史记录管理

- 点击首页"查看历史记录"

## 🎯 最佳实践

### 建筑设计流程
1. 用文生图探索不同风格方向
2. 手绘概念草图
3. 用图生图将草图转为效果图
4. 微调提示词优化细节
5. 下载最终版本用于展示

### 室内设计流程
1. 测量空间尺寸
2. 用文生图生成风格参考
3. 绘制简单平面/立面草图
4. 用图生图生成多角度效果图
5. 根据结果调整设计方案

### 提高工作效率
- 保存常用提示词模板
- 建立个人风格库
- 批量生成多个方案对比
- 利用历史记录追溯灵感

## ⚠️ 使用限制

1. **API 配额**: Replicate 按量计费，注意账户余额
2. **生成时间**: 高峰时段可能需要更长时间
3. **图片版权**: 生成图片的版权归属请查阅 Replicate 条款
4. **内容政策**: 不得生成违法、侵权或不适当内容

## 🆘 故障排查

### 问题：一直显示"正在生成中"
**解决：**
- 检查网络连接
- 确认 REPLICATE_API_TOKEN 正确
- 等待 60 秒，可能是队列拥堵
- 刷新页面重试

### 问题：生成结果不符合预期
**解决：**
- 增加描述细节
- 使用更专业的术语
- 尝试不同风格预设
- 调整 negative prompt

### 问题：图生图效果不明显
**解决：**
- 上传更清晰的草图
- 在描述中强调想要的变化
- 尝试不同的渲染风格
- 调整原图对比度
