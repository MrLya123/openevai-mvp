## 🚀 部署选项

### 选项 1：Vercel 部署（推荐）

**优点：**
- 免费、稳定、全球 CDN
- 自动 HTTPS
- 一键部署

**步骤：**
1. 将代码推送到 GitHub
2. 访问 https://vercel.com/new
3. 导入 GitHub 仓库
4. 配置环境变量 `REPLICATE_API_TOKEN`
5. 点击 Deploy

**预计时间：** 5 分钟
**结果：** 获得 `https://xxx.vercel.app` 公开链接

---

### 选项 2：下载压缩包手动部署

**适用场景：**
- 不想使用 GitHub
- 想部署到其他平台（Netlify、Railway 等）

**步骤：**
