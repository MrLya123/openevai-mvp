
访问 http://localhost:3000

### 2. 配置 API Token（如需真实生成）

```bash
# 编辑环境变量
nano .env.local
```

修改为：
```
REPLICATE_API_TOKEN=your_actual_token_here
NEXT_PUBLIC_DEMO_MODE=false
```

获取 Token: https://replicate.com/account/api-tokens

## 📋 功能详细说明

### 文生图功能

**输入：**
- 提示词（支持大白话）
- 风格选择（12 种预设）
- 图片尺寸

**处理流程：**
1. 用户输入大白话描述
2. 系统调用提示词转换助手
3. 生成专业级 AI 绘画提示词
4. 调用 Replicate SDXL API
5. 返回生成结果

**风格预设：**
1. 现代中式
2. 极简主义
3. 新古典
4. 工业风
5. 北欧风格
6. 新中式室内
7. 现代简约
8. 法式优雅
9. 意式轻奢
10. 日式侘寂
11. 奶油风
12. 自然生态景观

### 图生图功能

**输入：**
- 上传图片（草图/白模/照片）
- 渲染风格选择
- 提示词（可选）

**处理流程：**
1. 用户上传图片
2. 选择渲染风格
3. 调用 Replicate ControlNet API
4. 保持结构，改变风格/材质
5. 返回渲染结果

**渲染风格：**
1. 照片级渲染
2. 手绘风格
3. 水彩风格
4. 线稿风格

## 🔧 配置说明

### 环境变量

| 变量名 | 说明 | 必填 |
|-------|------|-----|
| REPLICATE_API_TOKEN | Replicate API 密钥 | 真实生成时必填 |
| NEXT_PUBLIC_DEMO_MODE | 演示模式开关 | 可选 |

### 演示模式

当 `NEXT_PUBLIC_DEMO_MODE=true` 时：
- 显示 UI 界面
- 展示提示词转换功能
- 生成操作显示提示（需配置 Token）

### 生产模式

当配置真实 Token 后：
- 完整功能可用
- 调用 Replicate API 生成图片
- 按量计费

## 🐛 常见问题

### Q: 为什么生成失败？
A: 检查 REPLICATE_API_TOKEN 是否正确配置，以及账户是否有足够额度。

### Q: 生成速度很慢？
A: Replicate 首次调用有冷启动时间（30-60 秒），后续调用会快很多。

### Q: 可以离线使用吗？
A: 当前版本需要联网调用 Replicate API。如需离线，需自建 ComfyUI 后端。

### Q: 如何添加自定义风格？
A: 编辑 `src/lib/prompts.ts` 文件，添加新的风格配置。
